# Assets Directory

## Images
Place your images in the appropriate subdirectories:
- family-photos/ - Family photographs
- slides/ - Kodachrome slides
- process-photos/ - Scanner, lightbox, working shots
- graphics/ - Diagrams and graphics
- placeholders/ - Generated by system

## Text
Place your essay text files here:
- portrait-essay.txt
- feature-article.txt
- reflection-essay.txt
- artist-statement.txt
- reflections.txt (150-word reflections for each assignment)

## Videos
- Put video embed codes/URLs in video-links.txt

## Docs
- Project briefing, assignment sheets, notes
